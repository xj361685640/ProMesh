External Libraries

Overview

ProMesh uses several closed- and open-source libraries. Below you'll find a list of all those libraries and the license under which each of them is used.

  • boost (Boost Software license)
  • lineintersect_utils (custom license)
  • linenoise (BSD-like license)
  • Lua (BSD-like license)
  • Qt (LGPL license)
  • RapidXML (Boost software license - Version 1.0)
  • TetGen (free for non-commercial use)
  • tribox (custom license)
  • tritri (custom license)
  • UG4 (custom license)
